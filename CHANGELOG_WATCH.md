# Changelog Watch

This file keeps the project from becoming stale.

## Sources To Watch

- OpenAI Codex product announcements
- OpenAI Codex developer docs
- `openai/codex` GitHub releases
- Codex CLI npm or package release notes
- Codex App feature documentation
- MCP, plugins, skills, hooks, automations docs

## Review Cadence

Weekly:

- check `openai/codex` releases
- scan Codex docs navigation for new pages
- check whether screenshots or UI-specific app guidance changed

Monthly:

- rerun self-assessment skill against the current docs
- update version matrix
- test all labs that execute commands
- verify all template paths and config examples

Before a tagged release:

- verify every chapter frontmatter
- run lesson quiz question review
- update official references
- mark deprecated examples
